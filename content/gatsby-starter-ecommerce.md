## Features

- Uses the Moltin e-commerce Api
- Stripe checkout
- Semantic-UI
- Styled components
- Google Analytics - (you enter the tracking-id)
- React-headroom
- ESLint & Prettier. Uses Airbnb JavaScript Style Guide
- Authentication via Moltin (Login and Register)
